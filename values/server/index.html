    Server

    dc_server

    Represents a server.

    Queryable properties #

    Property Type Description
    name String The name of the server.
    id String The ID of the server.
    members List of Members All users in this server.
    (Requires the GUILD_MEMBERS intent)
    users List of Users All users in this server.
    (Requires the GUILD_MEMBERS intent)
    channels List of Channels All channels in this server.
    roles List of Roles All roles in this server.
    webhooks List of Webhooks All webhooks in this server.
    Throws an exception on failure.
    slash_commands List of Slash commands All slash commands in this server.
    Throws an exception on failure.
    emojis List of Emojis All emojis in this server.
    sticker List of Stickers All stickers in this server.
